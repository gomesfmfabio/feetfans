# Contract Review Checklist

## Pre-Review
- [ ] Identificar tipo de contrato (prestacao de servicos, compra e venda, SPA, NDA, etc.)
- [ ] Identificar jurisdicao aplicavel
- [ ] Identificar partes envolvidas
- [ ] Verificar se ha modelo/template base

## Estrutura (Kenneth Adams - Categories of Contract Language)
- [ ] Clausulas de obrigacao ("shall") estao corretas
- [ ] Clausulas de proibicao ("shall not") estao presentes onde necessario
- [ ] Clausulas de discricao ("may") nao criam ambiguidade
- [ ] Clausulas de politica (declarativas) estao no lugar certo
- [ ] Definicoes sao consistentes em todo o contrato
- [ ] Cross-references estao corretas

## Deal Terms (Tina Stark - Five-Prong Framework)
- [ ] **Money**: Valor, forma de pagamento, prazos, multas
- [ ] **Risk**: Alocacao de risco clara entre as partes
- [ ] **Control**: Quem tem poder de decisao sobre o que
- [ ] **Standards**: Padroes de performance definidos
- [ ] **Endgame**: Condicoes de rescisao, saida, renovacao

## Linguagem (Bryan Garner - Plain English)
- [ ] Frases com menos de 25 palavras (media)
- [ ] Voz ativa predominante
- [ ] Sem legalese desnecessario
- [ ] Termos definidos usados consistentemente
- [ ] Sem ambiguidades ("e/ou", "incluindo mas nao limitado")

## Risco
- [ ] Clausulas de limitacao de responsabilidade adequadas
- [ ] Indenizacao bem delimitada
- [ ] Forca maior presente e atualizada
- [ ] Clausula de confidencialidade adequada
- [ ] Lei aplicavel e foro de eleicao definidos

## Compliance (se aplicavel)
- [ ] LGPD: Clausula de protecao de dados presente
- [ ] Anticorrupcao: Clausula de compliance presente
- [ ] Tributario: Retencao na fonte prevista
- [ ] Trabalhista: Nao caracteriza vinculo (se terceirizacao)

## Direito Brasileiro Especifico (se aplicavel)
- [ ] Funcao social do contrato respeitada (Art. 421 CC)
- [ ] Boa-fe objetiva observada (Art. 422 CC)
- [ ] Equilibrio contratual mantido
- [ ] CDC aplicavel? Se sim, clausulas abusivas verificadas
- [ ] Registro necessario? (imoveis, PI, etc.)

## Final
- [ ] Revisao por segunda pessoa recomendada para contratos de alto valor
- [ ] Versao final salva com controle de alteracoes
- [ ] Partes alertadas sobre riscos identificados
