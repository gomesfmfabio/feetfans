## Story Description

Implement an infinite scroll feed displaying content from all creators in reverse chronological order. Enable consumers to discover content with smooth scrolling, video autoplay, and clickable items that navigate to creator profiles.

---

## Acceptance Criteria

- [ ] **AC1:** Feed page at /feed (all authenticated users, default for consumers)
- [ ] **AC2:** Displays content in reverse chronological order (newest first)
- [ ] **AC3:** Each item shows image/video, creator nickname, category tags, timestamp
- [ ] **AC4:** Infinite scroll (20 items initially, fetch next 20 on scroll)
- [ ] **AC5:** Clicking item opens Creator Profile or full-screen view
- [ ] **AC6:** Video auto-plays when scrolled into view (muted by default)
- [ ] **AC7:** Loading spinner while fetching new content
- [ ] **AC8:** Empty state: "No content yet - check back soon!"
