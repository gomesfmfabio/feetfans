## Story Description

Implement age verification system requiring users to upload government-issued ID for 18+ verification. Integrate with Onfido API (or cheapest US-compliant alternative) to automatically verify age from uploaded documents. Block users under 18 immediately and update the `age_verified` flag in the database for users who pass verification.

This story ensures legal compliance with age restriction laws and protects the platform from liability related to minors accessing adult content.

## Technical Notes

### Age Verification Provider Research

**Option 1: Onfido**
- Pros: Robust API, global coverage, high accuracy
- Cons: Expensive (~$1-3 per verification)
- US-compliant: Yes

**Option 2: Persona**
- Pros: Cheaper than Onfido (~$0.50-1 per verification), modern API
- Cons: Less established, fewer integrations
- US-compliant: Yes

**Option 3: Vouched**
- Pros: Focused on ID verification, competitive pricing
- Cons: Smaller provider, less documentation
- US-compliant: Yes

**Option 4: Veriff**
- Pros: Good balance of price and quality
- Cons: Mid-tier pricing
- US-compliant: Yes

**Recommendation for MVP:** Start with **Persona** (cheapest compliant option) or **Onfido sandbox** (test only, no cost). Evaluate pricing and switch if needed.

### Supabase Storage Configuration

```sql
-- Create id-documents bucket
INSERT INTO storage.buckets (id, name, public)
VALUES ('id-documents', 'id-documents', false);

-- RLS policy: Users can only upload/read their own documents
CREATE POLICY "Users can upload own ID documents"
ON storage.objects FOR INSERT
WITH CHECK (
  bucket_id = 'id-documents' AND
  auth.uid()::text = (storage.foldername(name))[1]
);

CREATE POLICY "Users can read own ID documents"
ON storage.objects FOR SELECT
USING (
  bucket_id = 'id-documents' AND
  auth.uid()::text = (storage.foldername(name))[1]
);

-- Policy: Service role can access all documents (for admin review)
CREATE POLICY "Service role has full access to ID documents"
ON storage.objects
USING (bucket_id = 'id-documents' AND auth.role() = 'service_role');
```

### Age Verification Page Implementation

```typescript
// app/verify-age/page.tsx
'use client';

import { useState } from 'react';
import { useRouter } from 'next/navigation';
import { supabase } from '@feetfans/database';

export default function VerifyAgePage() {
  const router = useRouter();
  const [file, setFile] = useState<File | null>(null);
  const [uploading, setUploading] = useState(false);
  const [error, setError] = useState('');

  const handleFileChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    const selectedFile = e.target.files?.[0];
    if (!selectedFile) return;

    // Validate file type and size
    const validTypes = ['image/jpeg', 'image/png', 'application/pdf'];
    if (!validTypes.includes(selectedFile.type)) {
      setError('Please upload a JPEG, PNG, or PDF file');
      return;
    }

    if (selectedFile.size > 10 * 1024 * 1024) {
      setError('File size must be less than 10MB');
      return;
    }

    setFile(selectedFile);
    setError('');
  };

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    if (!file) return;

    setUploading(true);
    setError('');

    try {
      // Get current user
      const { data: { user } } = await supabase.auth.getUser();
      if (!user) throw new Error('Not authenticated');

      // Upload to Supabase Storage
      const filePath = `${user.id}/${Date.now()}-${file.name}`;
      const { error: uploadError } = await supabase.storage
        .from('id-documents')
        .upload(filePath, file);

      if (uploadError) throw uploadError;

      // Trigger Onfido verification via API endpoint
      const response = await fetch('/api/verify-age', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ filePath }),
      });

      if (!response.ok) throw new Error('Verification failed');

      // Poll for verification result or wait for webhook
      // For MVP: Show pending status and refresh page
      alert('Verification submitted! You will be notified when complete.');
      router.push('/verification-pending');
    } catch (err) {
      setError(err instanceof Error ? err.message : 'An error occurred');
    } finally {
      setUploading(false);
    }
  };

  return (
    <div className="min-h-screen flex items-center justify-center bg-gray-50">
      <div className="max-w-md w-full bg-white p-8 rounded-lg shadow-md">
        <h1 className="text-2xl font-bold mb-4">Age Verification</h1>
        <p className="text-sm text-gray-600 mb-6">
          To comply with age restrictions, please upload a government-issued ID (driver's license, passport, etc.).
          Your ID is encrypted and used only for age verification. We do not store your ID permanently.
        </p>

        <form onSubmit={handleSubmit} className="space-y-4">
          <div>
            <label className="block text-sm font-medium mb-2">Upload Government ID</label>
            <input
              type="file"
              accept=".jpg,.jpeg,.png,.pdf"
              onChange={handleFileChange}
              className="w-full"
            />
            {file && (
              <p className="text-sm text-gray-600 mt-2">
                Selected: {file.name} ({(file.size / 1024 / 1024).toFixed(2)} MB)
              </p>
            )}
          </div>

          {error && (
            <div className="text-red-600 text-sm">{error}</div>
          )}

          <button
            type="submit"
            disabled={!file || uploading}
            className="w-full bg-primary text-white py-2 rounded-md hover:bg-primary/90 disabled:opacity-50"
          >
            {uploading ? 'Uploading...' : 'Submit for Verification'}
          </button>
        </form>

        <p className="mt-4 text-xs text-gray-500">
          By uploading your ID, you confirm that you are 18 years or older.
        </p>
      </div>
    </div>
  );
}
```

### Backend API Endpoint for Onfido Integration

```typescript
// apps/api/src/routes/verify-age.ts
import { FastifyPluginAsync } from 'fastify';
import { supabase } from '@feetfans/database';
// import { Onfido } from 'onfido-node'; // npm install onfido-node

const verifyAgeRoute: FastifyPluginAsync = async (fastify) => {
  fastify.post('/verify-age', async (request, reply) => {
    const { filePath } = request.body as { filePath: string };

    try {
      // Get user from session
      const authHeader = request.headers.authorization;
      if (!authHeader) {
        return reply.status(401).send({ error: 'Unauthorized' });
      }

      // Verify user session
      const { data: { user }, error: authError } = await supabase.auth.getUser(
        authHeader.replace('Bearer ', '')
      );

      if (authError || !user) {
        return reply.status(401).send({ error: 'Invalid session' });
      }

      // Get file URL from Supabase Storage
      const { data: fileData } = supabase.storage
        .from('id-documents')
        .getPublicUrl(filePath);

      // TODO: Integrate with Onfido API
      // const onfido = new Onfido({ apiToken: process.env.ONFIDO_API_TOKEN });
      // const applicant = await onfido.applicant.create({ ... });
      // const check = await onfido.check.create({ applicantId: applicant.id, ... });

      // For MVP: Mock verification (simulate success after 5 seconds)
      setTimeout(async () => {
        // Simulate age verification result
        const isOver18 = true; // In production, this comes from Onfido

        if (isOver18) {
          await supabase
            .from('users')
            .update({ age_verified: true })
            .eq('id', user.id);
        } else {
          await supabase
            .from('users')
            .update({ age_verified: false, account_blocked: true })
            .eq('id', user.id);
        }
      }, 5000);

      return reply.send({ success: true, message: 'Verification in progress' });
    } catch (err) {
      fastify.log.error(err);
      return reply.status(500).send({ error: 'Verification failed' });
    }
  });
};

export default verifyAgeRoute;
```

### Onfido Webhook Handler

```typescript
// apps/api/src/routes/webhooks/onfido.ts
import { FastifyPluginAsync } from 'fastify';
import { supabase } from '@feetfans/database';
import crypto from 'crypto';

const onfidoWebhookRoute: FastifyPluginAsync = async (fastify) => {
  fastify.post('/webhooks/onfido', async (request, reply) => {
    try {
      // Verify webhook signature (Onfido security)
      const signature = request.headers['x-sha2-signature'] as string;
      const body = JSON.stringify(request.body);
      const hash = crypto.createHmac('sha256', process.env.ONFIDO_WEBHOOK_TOKEN!)
        .update(body)
        .digest('hex');

      if (signature !== hash) {
        return reply.status(401).send({ error: 'Invalid signature' });
      }

      const { payload } = request.body as any;
      const { object } = payload;

      // Check if verification is complete
      if (object.status === 'complete') {
        const applicantId = object.applicant_id;

        // Fetch applicant details from Onfido
        // const onfido = new Onfido({ apiToken: process.env.ONFIDO_API_TOKEN });
        // const applicant = await onfido.applicant.find(applicantId);

        // Extract age from document (DOB)
        const dob = new Date('2000-01-01'); // Mock DOB, in production from Onfido
        const age = new Date().getFullYear() - dob.getFullYear();

        // Update user record
        if (age >= 18) {
          await supabase
            .from('users')
            .update({ age_verified: true })
            .eq('id', applicantId); // Map applicantId to user.id
        } else {
          await supabase
            .from('users')
            .update({ age_verified: false, account_blocked: true })
            .eq('id', applicantId);
        }

        return reply.send({ success: true });
      }

      return reply.send({ success: true, message: 'Webhook received' });
    } catch (err) {
      fastify.log.error(err);
      return reply.status(500).send({ error: 'Webhook processing failed' });
    }
  });
};

export default onfidoWebhookRoute;
```

## Risks & Mitigations

**Risk:** Onfido API cost too high at scale (thousands of verifications per month)
**Mitigation:** Start with cheapest provider (Persona), evaluate volume discounts, consider switching providers if needed

**Risk:** Users upload invalid/fake IDs, bypassing age verification
**Mitigation:** Use reputable provider with document authenticity checks, flag suspicious verifications for manual review

**Risk:** ID documents stored permanently, violating data retention policies
**Mitigation:** Implement auto-deletion after 30 days, document retention policy in privacy policy

**Risk:** Webhook failures cause verification results not to update
**Mitigation:** Implement polling fallback (check Onfido API periodically), add retry logic for webhook failures

**Risk:** Legal compliance issues if age verification is not robust
**Mitigation:** Use US-compliant provider, document verification process, maintain audit logs for all verifications
