## Story Description

Create a public creator profile page displaying all content from a specific creator, profile information, and a "Message Creator" button. Allow consumers to browse a creator's portfolio before initiating contact.

---

## Acceptance Criteria

- [x] **AC1:** Creator profile page at /creator/[id] (all authenticated users)
- [x] **AC2:** Displays creator nickname, join date, total content count
- [x] **AC3:** Grid layout shows all creator's content (same as My Content)
- [x] **AC4:** "Message Creator" button navigates to chat (Epic 3)
- [x] **AC5:** Trial users see tooltip: "Upgrade to message real creators"
- [x] **AC6:** Content items viewable in full-screen lightbox/modal
- [x] **AC7:** 404 error if creator doesn't exist or account deleted

// app/creator/[id]/page.tsx
export default async function CreatorProfilePage({ params }: { params: { id: string } }) {
  const { data: creator } = await supabase
    .from('users')
    .select('nickname, created_at')
    .eq('id', params.id)
    .eq('role', 'creator')
    .single();

  const { data: content, count } = await supabase
    .from('content')
    .select('*', { count: 'exact' })
    .eq('creator_id', params.id);

}
