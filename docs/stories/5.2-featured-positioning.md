## Technical Notes

```typescript
// Feed query with featured priority
const { data: content } = await supabase
  .from('content')
  .select(`
    *,
    creator:users(
      id,
      nickname,
      featured:featured_placements(expires_at)
    )
  `)
  .order('creator.featured.expires_at', { ascending: false, nullsFirst: false })
  .order('created_at', { ascending: false });

// Cron job
cron.schedule('0 0 * * *', async () => {
  await supabase
    .from('featured_placements')
    .delete()
    .lt('expires_at', new Date());
});
```
