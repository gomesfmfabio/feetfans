## Story Description

Display all content uploaded by the authenticated creator in a responsive grid gallery. Allow creators to view their content portfolio, see engagement metrics (views - optional), and delete content items with confirmation.

---

## Acceptance Criteria

- [ ] **AC1:** My Content page at /my-content (creators only)
- [ ] **AC2:** Grid layout (2 cols mobile, 3-4 cols desktop)
- [ ] **AC3:** Each item shows thumbnail, category tags, upload date
- [ ] **AC4:** Delete button with confirmation modal
- [ ] **AC5:** Delete removes from database and Supabase Storage
- [ ] **AC6:** Empty state: "Upload your first photo or video to get started"
- [ ] **AC7:** Pagination or infinite scroll for >20 items
